### Configuration
- **General Settings:**
  - Enable product syncing either through scheduled tasks or directly in the admin panel:
    - Go to `Store > Configuration > Sales > Payment Methods > Stripe > Configuration > Recharge Payments Settings` (Turn on Debug Mode).
  - Make sure to enable Subscription for Recharge Payments to synchronize products.
